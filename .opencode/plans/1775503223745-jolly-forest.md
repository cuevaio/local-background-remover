# Safe Unused Code Cleanup

## Goal
Remove only high-confidence unused code in `apps/web`, and avoid compatibility, public API, experiment-config, or cross-app cleanup.

## Recommended Approach
1. Clean the two App Router pages that compute experiment attribution values but do not use them.
   - Modify `apps/web/src/app/page.tsx`
   - Modify `apps/web/src/app/downloads/page.tsx`
   - Remove the unused `mergeExperimentToken` and `readSingleParam` imports.
   - Remove the unused `resolvedSearchParams`, `incomingExp`, and `exp` locals.
   - If `searchParams` becomes unnecessary after that removal, simplify the page props and function signature in the smallest safe way.

2. Delete fully unreferenced web components.
   - Delete `apps/web/src/components/marketing/LogoStrip.tsx`
   - Delete `apps/web/src/components/ui/separator.tsx`
   - Delete `apps/web/src/components/ui/slider.tsx`
   - Before deletion during implementation, recheck for non-code references such as barrel exports, MDX imports, or dynamic imports.

3. Fix only direct fallout from those removals.
   - Update any broken imports surfaced by the build.
   - Do not expand scope into refactors, dependency pruning, or broader UI cleanup unless the build requires a tiny follow-up.

## Files In Scope
- `apps/web/src/app/page.tsx`
- `apps/web/src/app/downloads/page.tsx`
- `apps/web/src/components/marketing/LogoStrip.tsx`
- `apps/web/src/components/ui/separator.tsx`
- `apps/web/src/components/ui/slider.tsx`

## Explicit Non-Goals
- Do not remove anything in `apps/rmbg`.
- Do not remove anything in `apps/desktop`.
- Do not remove experiment flag definitions, attribution helpers, or Vercel Flags wiring outside the two page-level cleanups.
- Do not remove shadcn-style exported helpers like `buttonVariants`, `badgeVariants`, `tabsListVariants`, or `AlertAction` in this pass.
- Do not remove backward-compatibility paths, public IPC/API surfaces, or protocol handlers.

## Verification
1. Reference sweep before and after edits:
   - `rg "LogoStrip|Separator|Slider" apps/web/src`
   - `rg "readSingleParam|mergeExperimentToken" apps/web/src/app/page.tsx apps/web/src/app/downloads/page.tsx`

2. Validate the web app still compiles:
   - `cd apps/web && bun run build`

3. Run the broader repo check after the targeted web build passes:
   - `bun run check`

4. If validation fails, limit fixes to import/type fallout directly caused by this cleanup and rerun the same commands.
